# Work Experience

Source of truth for the `Experience` entries consumed by `app/data/en.ts` and `app/data/fa.ts`. Each company maps to a top-level `Experience` and each project under it maps to an entry in `children`. Field labels (`Title`, `Description`, `Skills`, etc.) mirror the names in the `Experience` interface in `app/data/types.ts`.
